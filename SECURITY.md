# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in any WorshipBuddy project, please report it responsibly. **Do not open a public issue.**

Email us at **support@worshipbuddy.org** with:

- A description of the vulnerability
- Steps to reproduce the issue
- Any relevant screenshots or logs

We will acknowledge your report within **48 hours** and work to resolve verified vulnerabilities as quickly as possible.

## Supported Versions

We provide security updates for the **latest released version** of each product. Older versions are not actively maintained.

| Product | Supported Version |
| --- | --- |
| PresenterBuddy  | Latest release |
| PresenterBuddy Studio | Latest release |
| ChurchBuddy (iOS/Android) | Latest release |
| WorshipBuddy API | Current production |

## Responsible Disclosure

We ask that you give us reasonable time to address the issue before disclosing it publicly. We are happy to credit researchers who report valid vulnerabilities.
