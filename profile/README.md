### About Us

WorshipBuddy is a **501(c)(3) nonprofit** building free, open-source tools for churches and worship teams. We believe every ministry — regardless of size or budget — deserves great technology.

### Our Products

| Product | Description |
| --- | --- |
| **[WorshipBuddy](https://worshipbuddy.org/worship-buddy)** | Song library, setlist management, and chord charts for worship leaders |
| **[ChurchBuddy](https://worshipbuddy.org/church-buddy)** | Mobile app for church scheduling, events, and team coordination |
| **[PresenterBuddy](https://worshipbuddy.org/presenter-buddy)** | Free lyric projection software for churches (Lite & Studio editions) |
